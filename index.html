Pinnacle 🏔️

Hackathon Semana Técnica — E.E. Padre Anchieta (2026)

Plataforma web de quizzes multiplayer em tempo real inspirada no Kahoot!. Criação de quizzes com até 20 perguntas, salas ao vivo com código PIN de 6 dígitos, pontuação com bônus por velocidade e ranking com pódio.

  • Criação de quizzes com timer e pontuação
  • Salas ao vivo com PIN de 6 dígitos
  • Multiplayer com HTTP polling
  • Pontuação com bônus por velocidade
  • Ranking e pódio dos 3 primeiros
  • QR Code para compartilhamento rápido
  • Upload de imagens nas perguntas
  • Autenticação com bcrypt e design responsivo
PHP MySQL (PDO) JavaScript Vanilla Fetch API QR Code Bootstrap Icons

API Multiplayer

API REST responsável pelo gerenciamento do estado das salas multiplayer, incluindo join, start, status, answer e advance.

  • Gerenciamento via arquivos JSON
  • Endpoints: CRUD de quizzes e salas
  • Comunicação assíncrona (HTTP polling)
  • Reset e reinício de partidas
PHP JSON REST

Banco de Dados

Modelagem relacional completa com usuários (alunos/professores), questionários, perguntas e alternativas.

  • 4 tabelas: usuario, questionarios, perguntas, answers
  • FK com CASCADE entre perguntas e answers
  • PIN único de 6 caracteres por quiz
  • Senhas hash (bcrypt) para autenticação
MySQL PDO phpMyAdmin
